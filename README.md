## 🎯 Sypixel RPG에 오신 것을 환영합니다!

Sypixel RPG는 마인크래프트에서 경험할 수 있는 가장 완성도 높은 RPG 서버입니다. 9개의 독특한 직업, 30개의 완성된 퀘스트, 개인 섬 시스템, 그리고 활발한 커뮤니티가 여러분을 기다리고 있습니다.

## 🚀 빠른 시작 가이드

### 1️⃣ 첫 접속시
```
1. 서버에 접속하면 튜토리얼이 자동으로 시작됩니다
2. /메뉴 명령어로 메인 메뉴를 열 수 있습니다
3. 튜토리얼을 완료하면 직업을 선택할 수 있습니다
```

### 2️⃣ 웹사이트 계정 연동
```
/사이트계정발급 your@email.com
```
- 웹사이트와 연동하여 더 많은 기능을 이용할 수 있습니다
- 자동으로 생성된 비밀번호를 클릭하여 복사할 수 있습니다

## 🏝️ 개인 섬 시스템 - 나만의 공간을 만들어보세요

### 섬 생성하기
1. `/섬` 명령어를 입력합니다
2. 섬 생성 GUI에서 다음을 설정합니다:
   - 🏷️ **섬 이름** (최대 20자)
   - 🎨 **이름 색상** (8가지 색상 중 선택)
   - 🌍 **바이옴** (평원, 숲, 사막, 설원, 정글, 늪, 사바나, 버섯 들판)
   - 🏗️ **기본 템플릿** (4가지 중 선택)

### 섬 관리 기능
- **👥 멤버 관리**: 친구를 초대하여 함께 건설
- **🔒 권한 설정**: 누가 무엇을 할 수 있는지 세밀하게 설정
- **📍 워프 포인트**: 빠른 이동을 위한 지점 설정
- **🔧 업그레이드**: 섬 크기 확장, 특별 기능 추가

- **🏆 업적 퀘스트**: 특별한 조건 달성시 완료 (10개 구현)

## 💰 경제 시스템 - 부자가 되어보세요

### 화폐 종류와 용도
| 화폐 | 아이콘 | 주요 용도 | 획득 방법 |
|------|--------|-----------|-----------|

2. **몬스터 사냥**으로 아이템을 획득하고 판매하세요
3. **채광**으로 광물을 모아 판매하세요
4. **섬 농장**을 만들어 작물을 재배하세요

## 👥 소셜 시스템 - 함께 즐기는 재미

### 친구 시스템
```
/친구추가 <닉네임> [메시지] - 친구 요청 보내기
/친구 - 친구 목록 확인
/친구삭제 <닉네임> - 친구 삭제
```

### 메일 시스템
```
/우편보내기 <닉네임> <제목> [내용] - 우편 발송 (아이템 첨부 가능)
/우편함 - 받은 우편 확인
```

### 귓속말
```
/귓속말 <닉네임> <메시지> - 개인 메시지
/w <닉네임> <메시지> - 단축 명령어
```

## 📊 스탯 시스템 - 캐릭터를 강화하세요

### 기본 스탯
| 스탯 | 약어 | 효과 | 추천 직업 |
|------|------|------|-----------|
| **근력** | STR | 물리 공격력 증가 | 전사, 궁수 |
| **지능** | INT | 마법 공격력, 마나 증가 | 마법사 |
| **민첩** | DEX | 공격 속도, 회피율 증가 | 궁수, 암살자 |
| **활력** | VIT | 체력, 방어력 증가 | 탱커, 전사 |
| **지혜** | WIS | 마나 재생, 스킬 쿨타임 감소 | 프리스트, 마법사 |
| **행운** | LUK | 크리티컬 확률, 아이템 드롭률 | 모든 직업 |

### 스탯 포인트 획득
- 레벨업마다 5 스탯 포인트 획득
- 특정 퀘스트 완료시 추가 포인트
- 이벤트 보상

## 🎯 플레이 팁 & 가이드

### 초보자 가이드
1. **튜토리얼 완주**: 기본 보상과 함께 게임 이해
2. **직업 선택**: 플레이 스타일에 맞는 직업 선택
3. **일일 퀘스트**: 매일 꾸준히 완료하여 성장
4. **친구 만들기**: 파티 플레이로 더 빠른 성장
5. **섬 건설**: 자원 생산 시설 우선 건설

### 중급자 가이드
1. **스탯 최적화**: 직업에 맞는 스탯 분배
2. **장비 강화**: 인챈트와 강화로 전투력 상승
3. **보스 레이드**: 파티를 구성하여 도전
4. **경제 활동**: 거래를 통한 재산 증식
5. **길드 가입**: 더 큰 커뮤니티 활동

### 고급자 가이드
1. **전설 퀘스트**: 최고 난이도 콘텐츠 도전
2. **PvP 참여**: 아레나에서 실력 겨루기
3. **시장 지배**: 아이템 거래로 경제 장악
4. **신규 유저 도움**: 커뮤니티 리더 되기
5. **콘텐츠 제작**: 유튜브, 스트리밍 활동

## ❓ 자주 묻는 질문

**Q: 직업을 바꿀 수 있나요?**
> A: 현재는 불가능합니다. 신중히 선택해주세요.

**Q: 섬을 여러 개 가질 수 있나요?**
> A: 플레이어당 1개의 섬만 소유 가능합니다.

**Q: 일일 퀘스트는 언제 초기화되나요?**
> A: 매일 오전 6시에 초기화됩니다.

**Q: 최대 레벨은 몇인가요?**
> A: 직업별로 다르며, 95~205 레벨입니다.

**Q: 친구는 몇 명까지 추가할 수 있나요?**
> A: 최대 100명까지 가능합니다.
